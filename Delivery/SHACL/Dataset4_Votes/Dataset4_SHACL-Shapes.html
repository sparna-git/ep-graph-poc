Vote

A Vote that took place within a sitting, identified with a vote number. Example URI \"/eli/dl/iPlMeeting/20191219/3\"
  • Target Class: eponto:Vote
  • Nodes: sh:IRI
  • URI: ^http://data.europarl.europa.eu/eli/dl/iPlMeeting/[0-9][0-9][0-9][0-9][0-9][0-9][0-9][0-9]/[0-9][0-9]?[0-9]?$
Property name URI Expected value Card. Description
vote title skos:prefLabel rdf:langString
1..1 Name of the voted text (from RollCallVote.Description.Text)
vote identifier eponto:voteIdentifier xsd:integer
1..1 Number of the vote (from RollCallVote.Result/@Number)
consists of MEP vote eponto:voteConsistsOfMEPVote MEP Vote
1..* Link to MEP votes in that Vote
forms part of plenary sitting eponto:voteFormsPartOfPlenarySitting Plenary Sitting
1..1 Link to plenary sitting during which the vote took place

MEP Vote

Represents the vote of a single MEP within a Vote. This is identified by the Vote number and the MEP ID. Example URI \"/eli/dl/iPlMeeting/20191219/3/MEP_342\"
  • Target Class: eponto:MEPVote
  • Nodes: sh:IRI
  • URI: ^http://data.europarl.europa.eu/eli/dl/iPlMeeting/[0-9][0-9][0-9][0-9][0-9][0-9][0-9][0-9]/[0-9][0-9]?[0-9]?/MEP_.*$
Property name URI Expected value Card. Description
mep eponto:voteByMEP MEP
1..1 Link to the MEP that voted
position eponto:votePosition Vote Position

(position:for, position:against, position:abstention)

1..1 Indicates whether to vote is « for », « against » or « abstention »
political group eponto:voteFromPoliticalGroup Political group
1..1 Link to political group to which the MEP belongs when the vote took place
forms part of vote eponto:MEPVoteFormsPartOfVote Vote
1..1 Link to the vote
