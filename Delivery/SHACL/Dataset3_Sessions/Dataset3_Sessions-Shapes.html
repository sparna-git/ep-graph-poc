Plenary Session

A Plenary Session spans multiple days and is identified with its begin date and end date. Example URI \" /eli/dl/iPlMeetingGroup/20191216-20191219\"
  • Target Class: eponto:PlenarySession
  • Nodes: sh:IRI
  • URI: ^http://data.europarl.europa.eu/eli/dl/iPlMeetingGroup/[0-9][0-9][0-9][0-9][0-9][0-9][0-9][0-9]-[0-9][0-9][0-9][0-9][0-9][0-9][0-9][0-9]$
Property name URI Expected value Card. Description
label rdfs:label xsd:string
1..1 Label is automatically generated with the begin date and end date of the session
begin date eponto:datePlenaryWeekBegin xsd:date
1..1 First day of this plenary session
end date eponto:datePlenaryWeekEnd xsd:date
1..1 Last day of this plenar session
consists of sitting eponto:plenarySessionConsistsOfPlenarySitting Plenary Sitting
1..* Reference to each sitting within that session
